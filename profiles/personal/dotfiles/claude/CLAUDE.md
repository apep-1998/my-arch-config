# Global instructions (personal profile)

## Python

Use `uv` for all Python work — dependency management (`uv add`, `uv remove`, `uv sync`, `uv lock`), running scripts and commands (`uv run`), creating virtual environments (`uv venv`), and installing standalone tools (`uv tool install`). Don't use `pip`, `pip install`, `python -m venv`, `pipx`, `poetry`, or `pdm` unless a project's existing tooling explicitly requires them.
